# Movie-Analysis
A data analysis project exploring key factors that influence a movie's box office success.

## Research Questions
1. Does a movie's budget affect its success?
2. Does the majority of profit come domestically or internationally?
3. Do shorter movie titles make more money?
4. Does release season affect box office performance?
5. Which genre has the most influence on a movie's success?

## Dataset
This project uses the [Movie Dataset for Analytics and Visualization](https://www.kaggle.com/datasets/mjshubham21/movie-dataset-for-analytics-and-visualization) from Kaggle.
